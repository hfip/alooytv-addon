# AlooYTV Stremio Addon

إضافة Stremio لمشاهدة المسلسلات والأفلام العربية والتركية والخليجية والأجنبية من موقع AlooYTV.

## الكتالوجات المتاحة

- أحدث الحلقات
- رمضان عربي 2026 ★
- رمضان خليجي 2026 ★
- مسلسلات تركية، عربية، خليجية، فارسية
- مسلسلات كورية، آسيوية، أجنبية
- أفلام كورية وأجنبية
- أنمي

## التثبيت السريع

الصق الرابط في قسم الإضافات بتطبيق Stremio:
```
https://YOUR_DOMAIN/manifest.json
```

## التشغيل المحلي

```bash
npm install
npm start
```

ثم أضف `http://localhost:7000/manifest.json` في Stremio.
